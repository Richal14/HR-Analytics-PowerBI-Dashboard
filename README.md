# HR Analytics Dashboard | Power BI

## Overview
This project analyzes employee data to understand attrition trends and workforce insights.

## Tools Used
- Power BI
- DAX
- Data Visualization

## KPIs
- Total Employees
- Attrition Count
- Attrition Rate %
- Average Salary
- Average Experience

## Analysis Performed
- Attrition by Department, Gender, and Job Role
- Salary and Experience impact on attrition
- Performance rating analysis

## Dataset
- HR_Analytics_Dataset.csv
